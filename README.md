# Dotfiles configuration
---

This dotfiles configuration is for Java and React development using WSL2 in windows

## Technologies:
- Neovim
- WSL: Ubuntu-24.04
- Git
- CLI: Allacritty, Wezterm, Terminal
- Shell: Zsh
- Tmux
- Lazygit
- Lazydocker


## Installation

```zsh
git clone git@github.com:HauHH1910/dotfiles.git
```
